# ez type

Un jeu de dactylographie rapide sur clavier AZERTY.

**Jouer** : [ez-type.vercel.app](https://ez-type.vercel.app/)

## Modes de jeu

- **15s / 30s / 60s** : tape un maximum de mots dans le temps imparti. Les erreurs comptent mais ne t'arrêtent pas.
- **Mort Subite** : pas de limite de temps (5 min max), mais la moindre erreur met fin à la partie.
- **ez** : comme Mort Subite, sauf que le mot à taper est toujours "ez".


## Score

Le score final en Mort Subite et ez est calculé avec : **mots corrects x WPM**.
